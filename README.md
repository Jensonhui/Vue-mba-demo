## vue-mba

> A Vue.js project

> 清华MBA移动端H5分享(个人练习)  

> http://mba.sem.tsinghua.edu.cn/sem_newsletter/mba/2020/


### Build Setup

``` bash
# 安装依赖
npm install

# 本地服务
1 . cd server 

2 . json-server --watch db.json

# 启动项目
npm run dev
```

### Main knowledge
1 . 路由切换，非父子关系兄弟组件传值

2 . html2canvas截图保存 填坑(跨域、图像模糊、toDataURL等)
